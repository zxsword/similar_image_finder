# 相似壁纸查找与清理工具 (Similar Image Finder)

这是一个基于 Python 的跨平台本地图库清理工具，旨在帮助用户极速找出并清理视觉上相似或重复的图片。无论是轻微的颜色调整、比例拉伸，还是增加了水印，它都能精准识别。

## ✨ 核心特性

- **双引擎智能检测**：
  - **增强视觉交叉引擎**：结合感知哈希 (pHash)、差异哈希 (dHash) 与色彩分布 (ColorHash)，无需庞大依赖即可杜绝大部分误报。
  - **AI 深度学习引擎**：按需自动加载 `MobileNetV2` 卷积神经网络，提取 1280 维高级语义特征，获得 Google 相册级别的防误判与抗裁剪能力。
- **极速本地缓存技术**：自动生成 `.sim_image_cache.json`，二次扫描相同或增量图库时，速度提升百倍。
- **现代化丝滑 GUI**：引入**分块异步渲染**与**队列懒加载**机制，即使检出上千组相似图片，界面滚动也绝不卡顿。
- **跨文件夹合并比对**：支持挂载辅助文件夹，轻松排查两个目录之间的冗余文件。
- **防误删保护**：采用 `send2trash` 机制，所有删除操作均将文件移至系统**回收站**，安全无忧。
- **极客与脚本友好**：提供完全解耦的命令行界面 (CLI) 支持，并支持多进程并发计算。
- **广度格式支持**：支持 JPG, PNG, WEBP 等主流格式，并可通过插件支持苹果 `HEIC` 格式。

## 🚀 安装指南

1. **克隆或下载本项目**到本地。
2. **安装基础依赖**：
   打开终端，在项目根目录下运行：
   ```bash
   pip install -r requirements.txt
   ```
3. **[进阶] 激活苹果 HEIC 格式支持 (可选)**：
   ```bash
   pip install pillow-heif
   ```
4. **[进阶] 激活 AI 深度学习引擎 (可选，推荐)**：
   安装 PyTorch 环境后，程序会自动感知并切换为 AI 模式：
   ```bash
   pip install torch torchvision
   ```

## 💻 使用说明

### 模式 1：图形用户界面 (GUI) - 推荐
直接双击运行 `main.py`（或在不带任何参数的情况下通过命令行启动）：
```bash
python main.py
```
- **滑动条调参**：支持实时拖动“相似度标准”滑块，数字越小越严格。
- **包含子文件夹**：灵活控制扫描深度。
- **查看 EXIF**：点击单张图片可直接查看完整元数据。

### 模式 2：命令行界面 (CLI) - 适合自动化脚本
在带参数的情况下运行 `main.py`，程序将直接进入静默处理模式：
```bash
# 仅扫描并输出报告（不移动文件）
python main.py "D:\MyPictures" --action report

# 扫描并把质量差的冗余图片移动到审核文件夹，同时设定较宽松的阈值 (10)
python main.py "D:\MyPictures" --action move --threshold 10

# 禁用子文件夹扫描，并指定使用 8 个进程加速
python main.py "D:\MyPictures" --no-recursive --workers 8
```

## 🧠 核心架构简述

本项目采用标准的 **领域驱动设计 (DDD)** 和 **MVC 架构**，非常适合 Python 新手作为进阶学习的参考：

- **根目录**：
  - `main.py`：程序的总入口，负责环境初始化，决定是启动图形界面还是命令行。
  - `gui_app.py`：主控制器 (Controller)，负责统筹调度界面组件和后台逻辑。
  - `find_similar.py`：纯命令行模式 (CLI) 的控制台，适合极客自动化调用。
- **`core/` (核心逻辑层)**：
  - `engine.py`：特征提取引擎。包含传统视觉哈希 (pHash) 和 AI 神经网络模型。
  - `matcher.py`：比对算法。负责计算指纹之间的“距离”并进行分组。
  - `cache.py`：缓存系统。通过保存 `.sim_image_cache.json` 加快二次扫描速度。
  - `scanner.py`：并发调度器。利用多线程(防界面卡死)和多进程(榨干CPU)执行扫描。
- **`ui/` (界面视图层)**：
  - 将庞大的界面拆分为了 `control_panel`(顶部)、`action_panel`(底部)、`preview_panel`(大图预览) 和 `pagination_view`(分页列表)。
- **`utils/` (通用工具层)**：
  - `file_ops.py`：负责各种操作系统级别的文件操作（如跨平台打开文件夹、安全移至回收站）。
